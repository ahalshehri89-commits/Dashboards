# Wiqaya (وقاية) Platform Agent Guide

## Project Overview
Wiqaya - One Health Platform for Disease Surveillance and Public Health Monitoring in Saudi Arabia

## Platform Purpose
**Wiqaya** (وقاية) - meaning "Prevention" in Arabic - is a comprehensive public health surveillance platform designed for Saudi Arabia's health authorities to monitor, track, and respond to disease outbreaks and health emergencies.

## Current Applications

### 1. Vaccine Calculator
- **File**: `/Vaccine Calculator/index.html`
- **Purpose**: Age-based vaccination schedule calculator
- **Features**:
  - Dual calendar support (Gregorian/Hijri)
  - Bilingual interface (Arabic/English)
  - Age calculation and vaccine recommendations
  - Responsive design with accessibility features

### 2. Dengue Surveillance Dashboards
- **Files**: `dashboard-v2.html`, `dashboard-momra.html`
- **Purpose**: Real-time dengue fever outbreak monitoring
- **Scope**: Multi-ministry surveillance system
